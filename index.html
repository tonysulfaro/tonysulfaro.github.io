Computer Science Coursework

CSE 231 - Python

This was my first programming class at Michigan State in Python. I learned quite a bit about data structures and ingesting data from a flat-file. The last two projects were games which were pretty cool because they utilized classes to store game data.

CSE 232 - C++

Overall, this is similar in structure to CSE 231 but in C++! One of the most challenging things I learned was how to use pointers and preventing memory leaks in my programs. Still the same flat file IO but using vectors and then with my own vector class and linked lists.

CSE 260 - Discrete Structures

This class gave me even more experience with computational theory including topics like data structures, algorithms, and finite state automata. Essentially the math behind all of the key CS concepts like state machines and set theory.

CSE 320 - Computer Organization and Architecture

How does a computer use circuits to store information? This and many other low-level concepts were emphasised in this course from low level system design to manipulating data within the ARM assembly.

CSE 331 - Algorithms and Data Structures

Who doesn't love some algorithms? This Python course focused solely on analyzing, designing, and implementing algorithms and making them more efficient. Some extra math is mixed in here as well.

CSE 335 - Object-Oriented Software Development

Utilizing C++ this class focuses solely on the design and implementation of large programs using OOP. This class focused mainly on the gang of four patterns (Abstract Factory, Builder, Adapter, Composite, Mediator, Observer, Template Method, Visitor)

CSE 477 - Web Application Development

By using PHP and some clinet-side Javascript I learned how to build and deploy PHP webapps to our CSE webdev server. Build tools like webpack and Sass were also used.

Human-Centered Design Coursework

MI 250 - Applied Programming

This course is outside of the college of engineering and focuses less on the depth of programming in Python more on the applications in the context of web development. I have a couple of projects on here utilizing Requests to interface with existing APIs and some others that extend the BaseHTTPRequestHandler to create an API and ones that also use Flask.

MI 320 - Reasoning with Data

This class focuses on deriving statistics and generating reporting in R through statistics such as means, standard deviations, contingency tables, and hypothesis tests. The first couple of labs are R script files while the rest are R Markdown files.

MI 349 - Web Design and Development

Focusing more on the design nuances of web development, this course gave me more experience in creating intuitive UX designs and making them accessible by writing semantically correct HTML.
